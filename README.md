## 本工具类可以做什么
 - Hash 处理。计算字符串 hash，文件 hash，获取 json token
 - Mongo 迭代返回查询结果
 - Mysql 迭代返回查询结果
 - Web 相关方法。增加文本的 soap 头，去除请求 url 中的无关参数
 - 通用方法。key 是否合法，点分割获取字典的值，迭代化节点的值，免去对类型的判断，是否为中文
 - 系统相关方法。返回本机地址
 - Date。时间相关方法，按不同方式返回一段区间的日期
 - Image。图片处理方法，是否为图片，图片是否完整，大小是否符合规范
 - More ......
